# Resources

These are the primary resources I used while preparing for the Microsoft Azure Administrator (AZ-104) certification. Every resource below contributed to my understanding in a different way, and together they provided a solid preparation strategy.

---

### 1. Video Course

My primary learning resource.

- John Savill AZ-104 [Playlist](https://www.youtube.com/playlist?list=PLlVtbbG169nGlGPWs9xaLKT1KfwqREHbs)
- John Savill AZ-104 [Study Cram](https://youtu.be/0Knf9nub4-k?si=c-Ttnxz92gTVmvTe)

The playlist explains the concepts extremely well, while the Study Cram is perfect for refreshing everything during the last 1–2 days before the exam.

---

### 2. Documentation

Microsoft's official learning material.

- Microsoft Learn [Course](https://learn.microsoft.com/en-us/training/courses/az-104t00)
- Microsoft Azure [Documentation](https://learn.microsoft.com/en-us/training/paths/az-104-administrator-prerequisites/)

Don't rush through Microsoft Learn. Whenever you struggle with a practice question or Azure service, come back here and read the official explanation.

---

### 3. Practice Exams

Practice exams were probably the biggest contributor to my success.

Two Udemy AZ-104 Practice Exam Courses:
- Udemy AZ-104 Practice ([Test Series #1](https://www.udemy.com/share/10cqqx3@zBx9_zNQ3iJtZyH8R-AOh6PR7Yu7LEfoJpLymx6pvD9ChqCKJ6s8UaQw0Z3nikw7/))
- Udemy AZ-104 Practice ([Test Series #2](https://www.udemy.com/share/10f1Mp3@e25MJuoWl3ghkQrsHCd1mmSXGLNcilDgjvpV_rT5jONnIXaOwhO_jK5KBlAL-N3q/))

Approximate totals:

- 12 Full-Length Practice Exams
- 900 Practice Questions

For each question, understand:

- Why the correct answer is correct
- Why the other options are wrong
- Which Azure service the question is actually testing

---

### 4. Hands-on Practice

Nothing replaces using Azure yourself.

- [Azure Portal](https://portal.azure.com/)
- Azure Free Trial subscription(s)
- Personal [lab deployments & documentations](../practice/HomeLabs.md)
- Build: Entra ID, Identities, Resource Groups, VMs, Storage, VNets, NSGs, RBAC, Azure Backup, Azure Monitor

Deploy -> Break -> Delete -> Rebuild -> Repeat.

---

### 5. AI-Assisted Learning

**I typically used AI to:**

- Explain difficult concepts
- Compare Azure services
- Review questions & answers for explanations
- Generate additional practice questions
- Create real-world scenarios & labs.
- Reinforce weak topics by repetition

### My Recommendation

If I had to prepare again, I'd follow almost the same approach:

1. John Savill Playlist
2. Microsoft Learn
3. Azure Portal (Hands-on)
4. Practice Exams
5. Review every incorrect answer / weak concept
6. John Savill Study Cram before the exam

---
