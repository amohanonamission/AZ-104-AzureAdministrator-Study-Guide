Things I learned

Read the last sentence first.

Microsoft exams usually ask for:

Most secure
Least administrative effort
Lowest cost
Highly available
Azure-native solution

Always identify that requirement first.

If two answers work,

choose the one that satisfies ALL requirements.

Know these really well:

RBAC vs Policy
Service Endpoint vs Private Endpoint
VPN Gateway vs ExpressRoute
Load Balancer vs Application Gateway
Traffic Manager vs Front Door
Availability Sets vs Availability Zones
Backup vs Site Recovery

Don't memorize.

Understand why each Azure service exists.
